

# 프로젝트 명

Number-Baseball-Game

# 프로젝트 기획 의도

- 멀티플레이에 대해 공부해보고 이를 기반으로 멀티게임 개발
- 클라이언트와 서버의 동기화에 중점을 두며 개발

# 프로젝트  소개

- 여러 명이서 즐길 수 있는 숫자 야구게임
- 라운드가 시작되면, 랜덤한 플레이어부터 정답을 입력할 수 있는 기회가 주어짐
- 5초안에 정답을 입력해야하며, 틀리거나 정답을 입력 안 했을 시 다음 플레이어에게 기회가 넘어감
- 이렇게 총 3바퀴를 돌며 중간에 정답을 맞춘 플레이어가 있다면 그 플레이어의 승리. 아무도 못 맞추었다면 무승부로 라운드 종료
- 라운도 종료 이후 새로운 플레이어가 있을 시 함께 다시 새 라운드 시작
